# GluteFitPro - Simplified Public Demo

A simplified, local-storage-only version of GluteFitPro for demonstration purposes.

## Features

- ✅ **Dashboard** - User profile management (stored in localStorage)
- ✅ **Workout Plans** - Generate personalized workout plans based on fitness level and goals
- ✅ **Meal Plans** - Get meal plan suggestions based on diet preferences
- ✅ **Local Storage Only** - No database, no authentication, all data stored in browser
- ✅ **Simple UI** - Clean, modern interface for workout and meal planning

## Removed Features

- ❌ Authentication & user accounts
- ❌ Database (Supabase/SQLite)
- ❌ 3D body visualizer
- ❌ Admin features
- ❌ Payment integration
- ❌ Advanced analytics
- ❌ Proprietary algorithms

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Data Storage

All data is stored in browser localStorage. No backend required.
